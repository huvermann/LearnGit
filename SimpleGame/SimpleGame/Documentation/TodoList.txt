  - 6.0: Beampoint
	  + 6.1: Tor mit xy-Koordinaten auf Map, damit man komfortabel von Map zu Map reisen kann.
         mit BeamPoint(x,y) soll man die Spielerposition innerhalb der Map konfigurieren.
		Bemerkung:

		Beampoints werden auf der Map im Objects-Layer angelegt.
		Jeder Beampoint muss einen eindeutigen Namen haben.
		Die xy-Position des Objekt-Rechtecks wird als Zielposition für den Spieler definiert
		über die globale Beampoint-Registry kann man alle Beampoints abrufen.
		Die beampoint.beam() methode erzeugt ein Change-View-Event. Die Positionsänderung wird dann vom Eventhändler erledigt.

		Jeder Beampoint im Spiel kann über die Beampoint-Registry angesprungen werden:
		
		Beispiel:
		beamService = ServiceLocator.getGlobalServiceInstance(ServiceNames.BeamPoints)
        beamService.beam("Ziel3")
